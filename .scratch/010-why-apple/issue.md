# Issue 10: Why Apple (Icon Card Gallery)

**Status:** `ready-for-agent`  
**Labels:** `feature`, `ready-for-agent`  
**Depends on:** Issue 1

## Description

Build the "Why Apple is the best place to buy AirPods" section with 5 icon cards that expand into modal overlays.

## Requirements

- Section header with "Shop AirPods" CTA
- Horizontal scroll gallery with 5 icon cards: Engraving, Payment, Delivery, Specialist, Apple Store App
- 56x56 icons + headline (28px) + body (17px)
- Expandable modal overlays with close button
- Card hover effects
- Padding nav for gallery navigation
- Light background (`background-alt`)
- Icons from Apple CDN

## Acceptance Criteria

- All 5 icon cards render with correct icons
- Horizontal scroll on all devices
- Each card opens its modal on click
- Modal close button is keyboard accessible
